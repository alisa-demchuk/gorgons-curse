# Medieval 2D Platformer (Project 222)

Прототип 2D экшен-платформера в средневековом сеттинге, разработанный на игровом движке **Godot 4**. Проект включает в себя базовые механики передвижения, боевую систему (в разработке), искусственный интеллект врагов и систему сбора предметов.

## 🎮 Основные возможности:

*   **Управление персонажем:**
    *   Передвижение и прыжки с физикой гравитации.
    *   Анимации персонажа (Idle, Run, Jump, Fall, Death) через `AnimationPlayer`.
    *   Заготовки под конечный автомат (State Machine) для атаки и блока.
*   **Искусственный интеллект (AI):**
    *   Враг "Горгона" (`Mobs/gorgon.gd`).
    *   Механика преследования игрока при попадании в зону видимости.
    *   Атака на ближней дистанции с нанесением урона.
*   **Игровой мир:**
    *   Параллакс-фон (`ParallaxBackground`).
    *   Тайловые карты (TileMapLayer) для построения уровней.
    *   Система переходов между уровнями через порталы.
    *   Смерть при падении ниже определенной глубины.
*   **Интерактивные предметы:**
    *   **Золото:** Сбор монеток с анимацией поднятия (Tween).
    *   **Зелья здоровья:** Восстановление HP игрока.
*   **Интерфейс (UI):**
    *   Главное меню с кнопками "Start" и "Exit".
    *   Отображение HUD (в процессе).
*   **Инструментарий:**
    *   Включен плагин **AS2P** (Animated Sprite to Player) для конвертации кадров `AnimatedSprite2D` в анимации `AnimationPlayer`.

## 🛠 Технический стек

*   **Движок:** Godot Engine 4.5 (Forward Plus Renderer).
*   **Язык:** GDScript.
*   **Ассеты:** Oak Woods (Brullov), спрайты рыцаря и горгоны.

## ⚙️ Установка и запуск

1.  **Требования:** Убедитесь, что у вас установлен Godot Engine версии 4.x.
2.  **Клонирование:**
    ```bash
    git clone https://github.com/your-username/medieval-blade-rpg.git
    ```
3.  **Импорт:**
    *   Запустите Godot.
    *   Нажмите "Import" и выберите файл `project.godot` в корневой папке проекта.
4.  **Запуск:**
    *   Нажмите `F5` для запуска игры (основная сцена настроена на меню).

## 📂 Структура проекта

*   `addons/` — Плагины редактора (включая AS2P).
*   `Collectibles/` — Скрипты и сцены подбираемых предметов (золото, зелья).
*   `Mobs/` — Сцены и логика врагов (Горгона).
*   `oak_woods_v1.0/` — Графические ассеты окружения.
*   `level_1.tscn`, `level_2.tscn` — Сцены уровней.
*   `player.gd` / `player.tscn` — Логика и сцена игрока.
*   `menu.tscn` — Главное меню.
